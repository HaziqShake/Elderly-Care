# Elderly-Care
An app made for caretakers of old age homes to track everything from meds, meals, health to hygiene and nutrition

after opening project folder-

# Install dependencies
npm install

# Navigation packages
npm install @react-navigation/native @react-navigation/bottom-tabs

# Extra dependencies for navigation
npm install react-native-screens react-native-safe-area-context react-native-gesture-handler react-native-reanimated react-native-get-random-values

# Vector icons (we used MaterialIcons)
npm install react-native-vector-icons @expo/vector-icons

# Supabase client (for backend later)
npm install @supabase/supabase-js

# Start Expo dev server
npm start
# (or: npx expo start)

# Then scan the QR code with the Expo Go app (Android/iOS)
